# [79_WordSearch.py 코드](./79_WordSearch.py) 예시

### 예시

- **보드판**: `[["A", "B"], ["C", "D"]]`
- **찾을 단어**: `"ABD"` (길이 3)
- **시작 지점**: `(0, 0)` 즉, 'A'부터 탐색 시작

### `_dfs` 예시 사례

#### **1단계: `_dfs(0, 0, 0)` 호출 (첫 글자 'A' 확인)**

1. `k=0` (단어의 첫 번째 글자 `"A"` 찾는 중)
2. `board[0][0]`는 `"A"`이고, `word[0]`도 `"A"`입니다. **(일치)**
3. `visited`에 `(0, 0)`을 추가합니다. `visited = [(0, 0)]`
4. 이제 다음 글자(`k=1`)인 `"B"`를 찾으러 상하좌우 탐색을 시작합니다.
* `res = _dfs(1, 0, 1) or _dfs(-1, 0, 1) or _dfs(0, 1, 1) or _dfs(0, -1, 1)`



#### **2단계: `_dfs(0, 1, 1)` 호출 (두 번째 글자 'B' 확인)**

* 위의 사방 탐색 중 `_dfs(0, 1, 1)`이 선택되었습니다. (오른쪽 칸 이동)

1. `k=1` (단어의 두 번째 글자 `"B"` 찾는 중)
2. `board[0][1]`은 `"B"`이고, `word[1]`도 `"B"`입니다. **(일치)**
3. `visited`에 `(0, 1)`을 추가합니다. `visited = [(0, 0), (0, 1)]`
4. 이제 마지막 글자(`k=2`)인 `"D"`를 찾으러 다시 상하좌우 탐색을 시작합니다.
* `res = _dfs(1, 1, 2) or _dfs(-1, 1, 2) ...`



#### **3단계: `_dfs(1, 1, 2)` 호출 (마지막 글자 'D' 확인)**

* `_dfs(1, 1, 2)`가 실행됩니다. (아래 칸 이동)

1. `k=2` (단어의 세 번째 글자 `"D"` 찾는 중)
2. `board[1][1]`은 `"D"`이고, `word[2]`도 `"D"`입니다. **(일치)**
3. `visited`에 `(1, 1)`을 추가합니다. `visited = [(0, 0), (0, 1), (1, 1)]`
4. 이제 다음 글자(`k=3`)를 찾으러 다시 호출합니다: `_dfs(..., ..., 3)`

#### **4단계: 종료 조건 발동 (`k=3`)**

1. `_dfs`가 호출되자마자 맨 첫 번째 줄인 `if k == len(word):`에 걸립니다.
2. `3 == 3` 이므로 **`True`를 반환**합니다.

#### **5단계: 백트래킹 (거슬러 올라가기)**

1. 3단계 함수가 `True`를 받고 `visited.pop()`을 실행해 `(1, 1)`을 지운 뒤 `True`를 리턴합니다.
2. 2단계 함수도 `True`를 받고 `visited.pop()`으로 `(0, 1)`을 지운 뒤 `True`를 리턴합니다.
3. 1단계 함수도 `True`를 받고 `visited.pop()`으로 `(0, 0)`을 지운 뒤 최종 `True`를 리턴합니다.
